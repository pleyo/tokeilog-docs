About This Repository

This repository contains the public website and documentation for TokeiLog, a privacy-focused time tracking app for iOS.

Website Structure

  • index.html - Main landing page with app overview
  • PRIVACY_POLICY.html - Privacy policy document
  • TERMS_OF_SERVICE.html - Terms of service agreement
  • LICENSE.html - Software license information
  • CHANGELOG.html - Version history and updates
  • APP_STORE_DESCRIPTION.html - App Store listing content
  • README.html - This documentation file

Development

This website is built with plain HTML and CSS for maximum compatibility and simplicity. No build tools or dependencies are required.
